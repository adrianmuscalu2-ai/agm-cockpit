# 🚛 AGM Translator

*AI-assisted translation and professional driver assistant for transport professionals.*

AGM Translator is an AI-powered assistant designed for truck drivers, transport companies, and logistics professionals. It helps users communicate across languages, prepare professional messages, and access transport-related tools from a single, intuitive interface.

## ✨ Key Features

- 🌍 Context-aware translation
- 🎤 Voice input and speech output
- ✍️ AI text correction
- 📧 Email Assistant
- 👤 User Profile
- 📷 OCR document recognition
- 🚚 Transport-focused AI assistance
- 🔒 Privacy-first approach

## 🎯 Mission

Reduce communication barriers in international transport by providing fast, accurate, and easy-to-use AI tools that improve safety, productivity, and efficiency.

## 🛠️ Technology

- TypeScript
- JavaScript
- NestJS
- Vue
- PostgreSQL
- OpenAI API
- OCR
- Android
- GitHub

## 🚀 Current Status

Development is actively progressing with continuous testing on both Android and Browser platforms.
